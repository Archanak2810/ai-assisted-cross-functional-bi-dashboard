# Data Quality and Scope Limitations

The dashboard applies practical checks including date conversion, duplicate handling, structural validation, valid-status filtering, and KPI reasonableness review.

Known limitations:

- Dataset is synthetic and demonstrates analytical capability rather than actual organisational results.
- Campaign spend-to-revenue comparison is a correlational proxy because orders do not contain a campaign identifier.
- Attrition is not calculated because employee exit-date data is unavailable.
- Predictive models are future scope; current AI-assisted insight generation is rule-based and explainable.
- Real-time pipelines, role-based access, automated alerts, and production governance require future implementation.
