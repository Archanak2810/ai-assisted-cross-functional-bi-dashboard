# GitHub and Streamlit Cloud Setup

Use `PUSH_TO_GITHUB.md` to publish the project to GitHub. Then follow `DEPLOY_TO_STREAMLIT_CLOUD.md` to create the Streamlit application.

Required files at repository root:

- `app.py`
- `requirements.txt`
- `.streamlit/config.toml`
- `src/bi_pipeline.py`

Main file path in Streamlit Cloud: `app.py`
