# Local Setup

## Requirements

- Python 3.10 or later
- VS Code or another code editor
- Git
- Optional: PostgreSQL/pgAdmin for local SQL exploration

## Steps

1. Create and activate a virtual environment.
2. Install dependencies using `pip install -r requirements.txt`.
3. Start the app with `streamlit run app.py`.
4. Either upload the RetailMart dataset ZIP or configure a local `.streamlit/secrets.toml` file for Neon.

The dashboard supports ZIP mode for a local demonstration and Neon PostgreSQL mode for cloud deployment.
