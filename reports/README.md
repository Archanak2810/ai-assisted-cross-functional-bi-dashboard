# Reports

This folder contains sample exported outputs from the dashboard Export tab.

## How to generate
1. Open the deployed Streamlit dashboard
2. Apply your desired filters in the sidebar
3. Navigate to the **Export** tab
4. Download:
   - `retailmart_management_report.txt` — Executive KPIs and AI insights summary
   - `retailmart_domainwise_bi_report.xlsx` — Multi-sheet Excel workbook with domain tables
   - `retailmart_management_report.html` — Shareable browser-ready report

Place the downloaded files in this folder before final GitHub push.
